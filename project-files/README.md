This folder containes:

1- molecular dynamics parameters for GROMACS 2020 software (.mdp files)
2- the folder named "0%" contains all input files of simulation of protein embedded in the membrane with 0% of oxidized POPC.
3- the folder named "25%" contains all input files of simulation of protein embedded in the membrane with 25% of oxidized POPC.
4- the folder named "50%" contains all input files of simulation of protein embedded in the membrane with 50% of oxidized POPC.
5- the folder named "75%" contains all input files of simulation of protein embedded in the membrane with 75% of oxidized POPC.
6- the folder named "100%" contains all input files of simulation of protein embedded in the membrane with 100% of oxidized POPC.
7- the folder named "Python-codes" contains all codes for analysis.

Any sub-folder has its own README file.
