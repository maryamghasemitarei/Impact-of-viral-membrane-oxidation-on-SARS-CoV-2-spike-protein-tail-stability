This folder containes all files related to 100% of POPC oxidation:
1- the folder named "toppar" contains the bonded, non-bonded parameters of atoms and the position restraints of proteins in charmm36 force field. 
2- the topology file named "topol". 
3- the index file named "index" contains some user definable sets of atoms.
4- the configuration file named "initial-structure" contains the coordinates of all atoms of system, including protein, glycans, membrane, water molecules and ions.

Any sub-folders has its own README file.
